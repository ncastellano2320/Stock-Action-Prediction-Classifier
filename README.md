# Stock-Action-Prediction-Classifier 
## Extending Logistic Regression 

This project deploys a Logistic Regression model using a One-vs-All (OvA) approach to predict whether market participants should buy, sell, or hold based on daily financial trends (open, high, low, and close prices). By analyzing historical market data, the model provides probabilistic insights to support informed trading decisions.

Built for use by financial institutions and trading platforms, this machine learning solution aims to enhance data-driven investment strategies.
